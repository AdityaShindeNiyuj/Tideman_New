#Tideman Problem:-
It is basically a ranked-choice voting system in which each voter can vote for more than one candidate according to their choice of preference.

#Sample Input:-
./tideman a b c
Number of voters: 5
Rank 1: a
Rank 2: c
Rank 3: b

Rank 1: c
Rank 2: a
Rank 3: b

Rank 1: b
Rank 2: c
Rank 3: a

Rank 1: b
Rank 2: a
Rank 3: c

Rank 1: c
Rank 2: a
Rank 3: b
